inicio en github Registrarse

Para comenzar a utilizar github es necesario registrarte en la página. Usa el formulario en pantalla o dale en en la parte superior derecha de la pantalla.

Paso 1. Crear cuenta personal.

El nombre de usuario sólo puede contener caracteres alfanuméricos o guiones simples, no debe tener espacios y no puede comenzar ni finalizar con un guion. La contraseña debe tener letras y números.

Paso 2. Elija su plan.

Elige el plan que deseas para utilizar tu cuenta github. Repositorios públicos ilimitados de forma gratuita o Repositorios privados ilimitados por $ 7 / mes. La diferencia será en que uno es de acceso público y el segundo privado.

Paso 3. Adapte su experiencia.

Para mejorar su experiencia utilizando github llene este formulario. Este último paso es opcional si lo desea puede omitir este paso. una ves finalizado entre a su correo para confirmar su cuenta.

Repositorio. creacion de repocitorios.

Para crear un repositorio en GitHub, En la esquina superior derecha, junto a tu avatar, haz clic en + y luego seleccione Nuevo repositorio.
1. Nombre su repositorio.
2. Escribe una breve descripción.
3. Seleccione Inicializar este repositorio con un README.

Repositorio. Crear archivos.

Para crear un archivo dentro del repositorio solo debe pulsar crear nuevo archivo y aparecerá un campo para escribir el nombre del archivo y para escribir su contenido.

Repositorio. agregar un colaborador.

Para añadir un colaborador diríjase a configuraciones , seleccionar colaboradores, y en la casilla de búsqueda escribir el nombre del colaborador.

Repositorio. Donde encontrarlos

Los repositorios que ya haya creado aparecerán en un recuadro en la pantalla principal.
